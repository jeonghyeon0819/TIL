## 📖 문제

### BOJ_1008_A/B
- 틀린 점 : float 사용으로 인해 소수점 정밀도가 부족하여 오차 발생.
- 해결 방법 : double로 형변환하여 계산
- 배운 점 : Java에서 실수 계산은 float보다 double이 더 높은 정밀도를 제공하기에 double을 사용해야 한다.

### BOJ_2588_곱셈
- 틀린 점 : 각 자릿수를 분리하지 않고 한 번에 계산하려고 해 문제의 요구 출력 형식을 맞추지 못함.
- 해결 방법 : AI를 활용하여 입력값의 일의 자리, 십의 자리, 백의 자리를 각각 나누어 계산하는 방식으로 수정.
- 배운 점 : 자릿수 분리는 나눗셈(/)과 나머지(%) 연산으로 처리할 수 있다.
