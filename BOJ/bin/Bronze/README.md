## 📖 단계별로 풀어보기 외 BOJ 브론즈 문제

### BOJ_1271_엄청난부자2
- 틀린 점 : long 타입으로 입력을 받았으나, 문제의 입력 범위(최대 (10^{1000}))를 처리할 수 없어 런타임 에러 발생.
- 해결 방법 : 큰 정수를 처리하기 위해 BigInteger 사용. / 나눗셈과 나머지 연산은 각각 divide(), remainder() 메서드로 수행.
- 배운 점 : Java에서는 long 범위를 초과하는 정수는 BigInteger로 처리해야 한다. / BigInteger는 연산자를 사용할 수 없고, 반드시 메서드를 통해 연산해야 한다.
